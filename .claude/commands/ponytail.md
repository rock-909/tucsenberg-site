Switch to ponytail $ARGUMENTS mode. If no level is specified, use full.

Lazy senior dev mode, before any code:

1. Does this need to exist at all?
2. Does the standard library already do it?
3. Does a native platform feature cover it?
4. Does an already-installed dependency solve it?
5. Can it be one clear line?
6. Only then, build the minimum that works.

No unrequested abstractions, no avoidable dependencies, no boilerplate.
Mark intentional simplifications with a `ponytail:` comment.
