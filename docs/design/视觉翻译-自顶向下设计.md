# 视觉翻译 · 自顶向下设计蓝图

> 2026-07-07 owner 校准后的需求定义（本文档是权威版本，取代此前的理解）：
>
> **"翻译"= 信息组织顺序 + 表达方式，必须从上到下设计（整站叙事 → 页面 → 屏 → 模块），
> 出发点是采购方的视角和问题。**
>
> 三个已确认的纠偏：
> 1. 翻译是**认知转换**（让不懂行的采购者顺着排版层级自然走完决策），不是媒介转换（加图）。
> 2. 设计要从**采购方的问题**里长出来，不是搬同行的版式模块。
> 3. 整站叙事逻辑先定，页面服从整站——自顶向下，不在单页内做加法。

## 一、采购方的问题序列（设计的唯一出发点）

所有买家进站时带着同一串问题，顺序基本固定：

| # | 买家心里的问题 | 出现时机 |
| --- | --- | --- |
| Q1 | 这家是干什么的？有没有我要的东西？ | 进站 0–10 秒 |
| Q2 | 哪个产品适合**我的**情况（场景/水型/地面/开口）？ | 首页→选型 |
| Q3 | 这东西本身靠谱吗（原理/规格/边界/对比便宜货）？ | 产品页 |
| Q4 | 买卖怎么做（MOQ/样品/价格逻辑/OEM/交期）？ | 产品页→OEM |
| Q5 | 这家**公司**靠谱吗（怎么验证/条款/是不是皮包）？ | 下单前 |
| Q6 | 下一步做什么？（摩擦要最小） | 全程 |

买家类型只改变问题的**权重**，不改变顺序：
- 经销商/批发：Q4 加重（利润、复购、不被绕过——"we don't retail"）
- OEM/品牌方：Q4 中的贴牌子项加重 → OEM 专页
- 承包商/项目：Q2、Q3 加重（规格证据、图纸、交期）
- 终端商业买家：Q2 加重（"我的车库进水了用哪个"）
- 技术评估方：Q3 加重（测试标准、数据、下载件）

## 二、整站叙事（页面 ← → 问题的映射）

```
首页        = Q1 + Q2 的翻译（是什么→五条线→怎么选→分流）
产品页 ×5   = Q2(单线深答) + Q3 + Q4 + Q6
OEM 页      = Q4 专线（贴牌/批发买家直达）
选型指南    = Q2 的深版（跨线对比）
规格指南    = Q3 的深版（技术评估方）
About       = Q5（验证方式）
Warranty    = Q5（条款）
RFQ/Contact = Q6（终点，摩擦最小）
页脚        = Q5 的常驻背书（法务全称+邮箱）
```

**动线检查标准**：任何一页的任何一屏，都必须能回答"买家此刻在问哪个 Q"；
答不出来的模块 = 装饰，删除或重排。

## 三、页面内的信息组织顺序（目标态）

### 产品页（以 TB-BW 为样板）

| 序 | 屏 | 回答 | 现状差距 |
| --- | --- | --- | --- |
| 1 | Hero（是什么+给谁+CTA）+ 数字条 | Q1 | ✅ 已就位 |
| 2 | 适合我吗：**场景墙 + 诚实边界** | Q2 | ⚠️ 场景墙目前在规格之后，应**提前**到 How it works 之前或合并；边界句散在 bullets 里 |
| 3 | 它怎么工作：原理（动画）+ 为什么不买便宜货（对比） | Q3 | ✅ 内容在，顺序对 |
| 4 | 规格表 | Q3 | ✅ |
| 5 | 怎么买：价格逻辑→小单/样品→OEM | Q4 | ✅ |
| 6 | 买多少：计算器 | Q4→Q6 | ✅ |
| 7 | FAQ（残余疑问） | Q3–Q5 | ✅ |
| 8 | CTA 强调带 | Q6 | ✅ |

**核心调整：场景墙从"规格后"提前到"Q2 位置"（How it works 之前），
因为买家先问"适合我吗"再问"它怎么工作"。**

### 首页

| 序 | 屏 | 回答 | 现状差距 |
| --- | --- | --- | --- |
| 1 | Hero：是什么+给谁+双 CTA | Q1 | ⚠️ hero 右侧"Product line map"与下方五卡重复索引（评审已指出）；hero 应回答 Q1，不预答 Q2 |
| 2 | 五条线（卡+线图） | Q1→Q2 | ✅ |
| 3 | 怎么选（决策表） | Q2 | ✅ 这是首页最该放大的差异化 |
| 4 | 怎么合作（流程五步 + 谁在供货 → OEM 入口） | Q4 | ✅ 顺序对 |
| 5 | 信任（验证方式摘要 → About） | Q5 | ⚠️ 目前首页缺 Q5 摘要，验证叙事只在 About/OEM |
| 6 | FAQ + CTA | Q6 | ✅ |

### 表达方式选型表（翻译的"文体"，全站统一）

| 信息类型 | 表达方式 |
| --- | --- |
| 决策/对比 | 表格（情境→答案两列优先） |
| 流程 | 编号步骤（数字标记） |
| 规格 | 全宽表 +（素材后）标注尺寸图 |
| 诚实边界 | 独立醒目块（leadNote 样式），不埋进 bullets |
| 可信证据 | 数字条（可核实白名单）、证书图位 |
| 原理 | 图/动画为主角，文字为标注 |
| 场景适配 | 场景墙（图位+一句适配说明） |
