# 设计

这里是 Tucsenberg 当前站的设计入口。

- `设计真相.md`：当前页面设计方向和 English-only 边界。
- `组件治理.md`：组件治理规则。
- `组件使用手册.md`：人和 agent 选择 UI wrapper 的短手册。
- `组件索引.md`：完整 UI wrapper inventory。
- `色彩系统.md`、`设计令牌.md`、`网格系统.md`：视觉系统参考。
- `动效治理.md`、`动效原则.md`：motion 边界。
- `页面模式.md`、`区块重设检查清单.md`：页面和区块重设参考。
- `Storybook覆盖范围.md`、`Storybook警告基线.md`：Storybook 相关证明。

旧原型、外部设计参考和 `docs/superpowers/specs/**` / `docs/superpowers/plans/**` 里的过程材料不能覆盖当前运行态。
