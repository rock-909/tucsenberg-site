## 一句话理解

- 这不是“纯前端展示站”。
- 它本质上是一个 **带多语言内容系统、留资处理链、安全防护层、Cloudflare 部署链、以及大量自检脚本的 B2B 官网平台**。

## 最关键的主链路

```text
买家访问页面
  -> 中间件处理语言与安全头
  -> 页面渲染组件与内容
  -> 用户提交联系/询盘表单
  -> 校验 + 限流 + Turnstile + 幂等保护
  -> 进入统一 Lead Pipeline
  -> 写入 Airtable / 发送 Resend 邮件
  -> 记录日志、指标、缓存和健康状态
```
