# Architecture Tree

Short map of the current repo.

```text
browser
  -> Cloudflare Worker / OpenNext
  -> middleware: retired-locale fast-404 + next-intl routing
  -> Next.js App Router
  -> localized pages
  -> components + content + messages
  -> inquiry, CSP report, and health APIs
  -> validation + rate limit + Turnstile
  -> lead pipeline
  -> Airtable / Resend
```

Main source areas:

- pages: `src/app/[locale]/**`
- components: `src/components/**`
- config: `src/config/**`
- content: `content/pages/**`
- messages: fixed `messages/base/**` -> `messages/profiles/b2b-lead/**` -> `messages/profiles/catalog/**`
- lead pipeline: `src/lib/lead-pipeline/**`
- deployment: `open-next.config.ts`, `wrangler.jsonc`

Security headers are owned by `next.config.ts` and `src/config/security.ts`,
not by middleware.
