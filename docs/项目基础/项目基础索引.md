# 项目基础索引

这里放项目机制和维护边界，不放长 workflow。

## 常用入口

- `项目基础.md`：当前 Tucsenberg 项目边界。
- `技术栈.md`：技术栈、Cloudflare/OpenNext、cache、CSP 和依赖策略。
- `部署.md`：Cloudflare、环境变量、表单和部署配置。
- `内容.md`：内容、SEO、messages 和内容检查。
- `monitoring.md`：Cookie、GA4、Search Console、Airtable 和流量监控阶段动作。
- `品牌.md`：品牌、域名和联系信息。
- `配置.md`：配置真相源和运行时 facade 边界。
- `行为合约.md`：用户可见行为合同。
- `维护规则.md`：owner map、truth registry、guardrails 和结构共改簇。
- `文档清单.md`：`docs/` 文件和目录为什么存在。

## 低频参考

- `架构图.svg` / `架构树.md`：架构图和文字 fallback。
- `集成.md`：候选集成 backlog，不是已批准接入清单。
- `派生配置.md`、`派生起步.md`、`替换顺序.md`、`替换边界.md`：继承 starter/profile 边界。
